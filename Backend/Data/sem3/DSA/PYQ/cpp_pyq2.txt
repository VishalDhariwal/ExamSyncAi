SECTION A [5 × 2 Marks = 10 Marks]
A1) List any four features of C++ that make it different from C. [2 Marks]
A2) What are built-in data types? List any five and mention their typical size in bytes (on most systems). [1+1 Marks]
A3) Explain relational and logical operators with one example each. [1+1 Marks]
A4) What is a destructor? Write its syntax and explain when it is called. [1+1 Marks]
A5) Differentiate between array and string in C++. Give declaration syntax for both. [1+1 Marks]
SECTION B [4 × 3 Marks = 12 Marks]
B1) Write a C++ program using for loop to print the sum of first n natural numbers (n input by user). [3 Marks]
B2) Explain encapsulation. Write a class Student with private members rollNo and marks, public methods setData() and getMarks(). Demonstrate data hiding. [1.5+1.5 Marks]
B3) What are jump statements? Explain break and continue with one loop example each showing their effect. [1.5+1.5 Marks]
B4) Write code to declare a C++ string object, assign “Bennett University”, concatenate “ C++”, and print its length. [3 Marks]
SECTION C [2 × 4 Marks = 8 Marks]
C1) Demonstrate operator overloading. Create class Time (hours, minutes), overload + operator to add two Time objects and return a new Time object. Write the class and show usage. [4 Marks]
C2) Explain dynamic memory allocation. Write a program that allocates an array of 5 floats using new[], inputs values from user, calculates average, and deallocates using delete[]. [2+2 Marks]
SECTION D [2 × 5 Marks = 10 Marks]
D1) Explain multiple inheritance with an example. Create classes Engine (power) and Body (weight), derive Car from both, add member price, and write display() to show all. Handle constructor calls properly. [5 Marks]
D2) What is STL? Explain any three containers (vector, map, set) with one line each. Write a program using vector<string> to store 4 names, sort them using algorithm, and print using iterator. [2+3 Marks]
