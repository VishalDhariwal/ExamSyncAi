SECTION A [5 × 2 Marks = 10 Marks]
A1) Who developed C++ and in what year was it first released? [2 Marks]
A2) List any four applications of C++ and explain one in brief. [2 Marks]
A3) Differentiate between #include and using namespace std; with one example each. [1+1 Marks]
A4) What is the difference between const keyword and #define preprocessor directive? Give one example for each. [1+1 Marks]
A5) Write the syntax for declaring a reference variable and a pointer variable to an integer. Explain one key difference between them. [1+1 Marks]
SECTION B [4 × 3 Marks = 12 Marks]
B1) Explain function overloading with an example. Write two overloaded functions named display – one that takes an integer and prints it, another that takes a string and prints it. Show how they are called. [3 Marks]
B2) What is a constructor? Explain default and parameterized constructors with syntax and one example each for a class Car. [1.5+1.5 Marks]
B3) Write a C++ code snippet using switch statement to accept a character (A, B, C, D, F) and print the corresponding grade description (Excellent, Good, Average, Poor, Fail). Include default case. [3 Marks]
B4) Explain increment and decrement operators with postfix and prefix forms. What will be the output of the following code segment?
int x = 5;
cout << x++ << " " << ++x << " " << x--; [1+2 Marks]
SECTION C [2 × 4 Marks = 8 Marks]
C1) Write a complete C++ program to demonstrate single inheritance. Create base class Vehicle with members speed and color (private), public method showDetails(). Derive class Bike that adds member mileage and overrides showDetails() to display all. Create object and call the method. [4 Marks]
C2) Explain exception handling using try, catch, throw. Write a program that takes two integers as input, divides them, and throws an exception if divisor is zero. Catch the exception and print “Division by zero error!”. [2+2 Marks]
SECTION D [2 × 5 Marks = 10 Marks]
D1) Design an abstract class Shape using abstraction with pure virtual function area(). Derive classes Triangle (base, height) and Square (side) that implement area(). Write complete code to create objects using base pointer and call area() polymorphically. Explain run-time polymorphism. [5 Marks]
D2) Write a function template swapValues<T> that swaps two values of any type. Also write a class template Pair<T1, T2> that stores two values of different types and has a member function display(). Demonstrate usage with int-float and string-double pairs. [2+3 Marks]
